Hexo+GitHub搭建博客

##Hexo+GitHub搭建博客

###流程
1 下载Node.js
2 考虑到npm国内比较慢可以去淘宝下cnpm:npm install -g –registry=https://registry.npm.taobao.org
3 cnpm下hexo: cnpm install -g hexo-cli
4 hexo搭建: 选择创建文件夹Blogs;初始化blog:hexo init;启动hexo start;创建文章hexo new “”
5 部署到GitHub: GitHub创建库;cnpm安装部署git:cnpm install –save hexo-deployer-git;_config_yml配置下type git,repo地址填入;最后部署 hexo d
6 主题的选择:找喜欢的去对应的GitHub clone到blog目录下的themes目录名字与——config.yml下themes后的名字一致
